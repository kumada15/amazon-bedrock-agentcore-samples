# Amazon Bedrock AgentCore コードインタープリターでのコマンド実行 - チュートリアル

## 概要

このチュートリアルでは、Amazon Bedrock AgentCore コードインタープリターを使用してコマンド（シェルおよび AWS CLI）を実行する方法を示します。AWS サービス、特に S3 操作に焦点を当てて説明します。以下を学びます：

1. Python ベースのコードインタープリターの作成
2. コードインタープリターセッションの開始
3. コマンドの実行（シェルおよび AWS CLI）
4. S3 操作の実行（バケット作成、オブジェクトコピー、バケットオブジェクトリスト）
5. クリーンアップ（セッション停止およびコードインタープリター削除）

### チュートリアルアーキテクチャ

コード実行サンドボックスは、コードインタープリター、シェル、ファイルシステムを備えた隔離環境を作成することで、ユーザークエリを安全に処理できます。大規模言語モデルがツール選択を支援した後、このセッション内でコードが実行され、ユーザーまたはエージェントに返されて統合されます。
