# 前提条件: サンプルエージェントの作成

## 概要

エージェントを評価する前に、評価対象となるエージェントが必要です。このチュートリアルでは、残りの評価チュートリアル全体で使用する2つのサンプルエージェントをセットアップします。1つは [Strands Agents SDK](https://strandsagents.com/) を使用し、もう1つは [LangGraph](https://www.langchain.com/langgraph) を使用します。

## エージェント
作成されるエージェントは基本的に同じですが、AgentCore の「任意のフレームワーク」という提案を示すために2つの異なるフレームワークを使用しています。

作成されるエージェントには2つの主要な機能があります：

**コード実行**
- AgentCore Code Interpreter を使用して Python コードを実行
- 数学計算とデータ分析を処理

**メモリ**
- ユーザーの事実と好みを保存
- パーソナライズされた応答のために関連するコンテキストを取得

両方のエージェントは LLM モデルとして Amazon Bedrock の Anthropic Claude Haiku 4.5 を使用しますが、AgentCore では好みの任意のモデルを使用できます。

## 前提条件
エージェントをデプロイする前に以下が必要です：
* Python 3.10 以上
* AWS アクセス
