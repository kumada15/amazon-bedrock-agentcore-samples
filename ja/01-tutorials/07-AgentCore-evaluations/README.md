# 概要

Amazon Bedrock AgentCore Evaluations は、実際のインタラクションに基づいてエージェントの品質を最適化するのを支援します。

## 主な機能

AgentCore Observability がエージェントの稼働状況に関する運用的な洞察を提供するのに対し、AgentCore Evaluations はエージェントの意思決定の質とパフォーマンス結果に焦点を当てます。

オンデマンドおよびオンライン評価機能を備えた組み込みおよびカスタム評価器を提供します。

### 組み込みおよびカスタム評価器

AgentCore Evaluations は、正確性、有用性、安全性などの重要な側面に対する13の組み込み評価器と、ビジネス固有の要件に対するカスタム評価器を作成する機能を提供します。

開発およびデプロイ中にオンデマンド評価APIを使用してエージェントをテストするか、オンライン評価APIで本番エージェントを監視します。

### オンデマンド評価

組み込みおよびカスタムメトリクスを使用して、個々のトレースに対して同期的なオンデマンド評価を実行します。

システムはOpenTelemetry（OTEL）トレースを使用してスコアリングを行い、以下を含むレスポンスを返します：
- スコア値
- スコアの説明
- トークン使用量

### オンライン評価

本番環境では、各トレースを手動で評価することなく、すべてのインタラクションにわたる継続的なパフォーマンス監視が必要です。統計的なサンプルは、意味のあるパフォーマンスメトリクスを生成するのに十分なことがよくあります。

AgentCore Evaluations のオンライン機能は、自動サンプリングと評価を可能にします：

- サンプルサイズとトレース選択基準を定義
- 評価メトリクス（組み込みまたはカスタム）を選択
- AgentCore Evaluations が残りを処理し、エージェントを大規模に監視するために必要なパフォーマンスデータを生成

## チュートリアル概要

これらのチュートリアルでは、以下の機能をカバーします：
- [前提条件](00-prereqs)：評価チュートリアルで使用するサンプルエージェントの作成
- [カスタム評価器の作成](01-creating-custom-evaluators)：組み込みおよびカスタムメトリクスについて学び、エージェントを評価するためのカスタムメトリクスを作成
- [オンデマンドおよびオンライン評価の使用](02-running-evaluations)：オンデマンドおよびオンライン評価を使用して、エージェントを構築、最適化、大規模に監視する方法を学ぶ
- [高度な機能](03-advanced)：boto3 SDK を使用してオンデマンド評価のための Amazon CloudWatch ログをクエリし、異なるエージェント設定での実験を可視化するローカルダッシュボードを作成するなど、高度な機能を探索
